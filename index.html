How It Works

Simple process for professional environmental testing

Mobile App Guide

  • Login using username & password.
  • Dashboard shows today’s tasks, monthly schedule, and sampling categories.
  • Choose any sampling type (Water, Air, Noise, Food, Waste, etc.).
  • Tap Add New Sample to start a new record.
  • Fill basic details (date, samples count, customer, etc.).
  • Take concerned person’s signature and continue.
  • Fill detailed sampling form (location, equipment ID, procedure, timings).
  • Add sampling images if required.
  • Preview the entire form.
  • Add sampling person’s signature and submit.
  • Use Draft List to edit incomplete forms.
  • Use Completed List to view submitted samples.

Admin Panel Dashboard

  • Admin & Managers login securely using credentials.
  • Dashboard shows stats like total Managers, Lab Assistants & Sampling Persons.
  • Manage Managers, Lab Assistants & Sampling Persons with add/edit options.
  • Create and manage Monthly Schedules for sampling persons.
  • Add and manage Customers for each city/branch.
  • Manage Branch Offices/Cities (Super Admin only).
  • View all sample categories and detailed sampling records.
  • Open Field IDs to view full sampling forms & download PDF.
  • Filter and view Sample Reports by customer, sample type & date.
  • View Scheduled Reports for sampling activities.
  • Update profile & change password through Account Settings.

Backend System Methodology

  • Centralized backend for managing users, customers & schedules.
  • Stores all sampling data submitted from mobile app.
  • Role-based access for Super Admin, Managers & Lab Assistants.
  • Generates Field IDs and organizes samples by category.
  • Secure cloud database with automatic data syncing.
  • Allows PDF generation for every sampling record.
  • Provides filtering options by date, customer & sample type.

Sample & Schedule Reports

  • Sample Reports:
  • View all submitted sample reports by lab assistants & sampling persons.
  • Filter reports by customer, sample type, city, or date range.
  • Generate detailed sample records with sampling person & customer info.
  • View sample categories (Water, Air, Stack Monitoring, etc.).
  • Track hazardous material handling via Hazarge Name.
  • View full details using action buttons for every record.

  • Schedule Reports:
  • Generate reports based on scheduled sampling activities.
  • Select sampling person or view their schedule collectively.
  • Filter schedule reports using From Date and To Date.
  • Display sampling person, manager name, and city in search results.
  • View customers & sample types collected during selected dates.
  • Managers can only view schedules of sampling persons in their own city.
  • Super Admin has access to all scheduled reports across all locations.
